

## Description

Backend api for a to do/scheduling application.

## Installation

```bash
$ npm install
```

## Running the app

First you need to create a local postgres database. An .env-example file is provided with all variables needed.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
